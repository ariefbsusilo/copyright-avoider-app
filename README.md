# Copyright Avoider App - Untuk Pembelajaran

Aplikasi Flutter edukatif untuk memahami teknik-teknik modifikasi konten yang sering digunakan untuk menghindari deteksi copyright di YouTube.

## ⚠️ DISCLAIMER
Aplikasi ini dibuat **HANYA UNTUK TUJUAN PEMBELAJARAN DAN EDUKASI**. 
- Tidak mendorong pelanggaran copyright
- Pengguna bertanggung jawab atas penggunaan konten
- Selalu gunakan konten yang Anda miliki haknya atau konten bebas royalti

## Fitur

1. **Video Modifier**
   - Ubah kecepatan video (0.5x - 2x)
   - Flip horizontal/vertical
   - Tambah filter warna
   - Crop dan resize

2. **Audio Modifier**
   - Ubah pitch audio
   - Ubah kecepatan audio
   - Adjust volume
   - Tambah noise/distortion

3. **Tips & Guidelines**
   - Panduan copyright YouTube
   - Sumber konten legal dan gratis
   - Best practices untuk creator

## Instalasi

```bash
flutter pub get
flutter run
```

## Persyaratan
- Flutter SDK 3.0+
- Android Studio / VS Code
- Emulator atau device fisik

## Cara Menggunakan
1. Pilih file video/audio dari perangkat
2. Pilih modifikasi yang ingin diterapkan
3. Preview hasil modifikasi
4. Export hasil ke galeri
